「ハムハムハムスターズ」は、1999年発売の PC 用ゲームソフト。PC 上でハムスターを育てられるゲームだ。ハム太郎が大人気になる以前、どちらかというと「ポストペット」に影響を受けたような飼育ゲーム。

自分は2020年にこのソフトの CD-ROM を買い直したが、Windows10 環境ではインストールすらできなかった。そこで以下のブログ記事にしたとおり、VMware を使って仮想 Windows 98 環境を立て、そこにハムハムハムスターズをインストールすることにした。

  • ブログ記事 : 2020-10-21 古い Windows アプリを動かすために VMware で Windows 98 環境を構築する

インストール後の exe ファイル群をホストマシンに持ち出せば、その後は Windows10 でも動作したが、ケージ内に水飲み機などを設置できない不具合がある他、そもそもハムスターの動く速度がゴキブリ並みに速くなる問題が起きていてまともにはプレイできない。ｗ

しかし、画面 UI の雰囲気や BGM は懐かしく、時々なんとなくハムスターを眺めている。ｗ
