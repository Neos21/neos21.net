    スーパーマリオアドバンス3 について

    発売日

    2002-09-20

    ジャンル

    アクション

    値段

    ゲーム紹介

    SFC の大人気ソフト、ヨッシーアイランドのリメイク版。これも前作同様、マリオブラザーズが付いてきて、通信対戦ができる。

    スーパーマリオアドバンス3 レビュー

    何かパッケージの質感が違うなぁと思っていたら、光沢のないマット紙を使っていたのね。手作り感を出させるためのようだ。手触りがよくてずっとパッケージを触っていたw。ゲームも手作り感が出ていて、暖か味があってとても良いゲームだ。

    スーパーマリオアドバンス3 裏技・小技

    おまけのスターマーク

    ステージ1からステージ8を100点にすると、タイトル画面のタイトルの文字の上にスターマークが出現する。

    反動

    棘の上からヒップドロップすると死んだとき反動が付いている。

    ビックパックン一撃

    ビックパックンの乗っている橋の丸いものの最初の1つ目に乗ってパックンが見えるくらいに進んで卵を当てると、カメックが来てパックンが死ぬ。

    踏ん張らなくても…

    秘密6の強制スクロール後の卵だらけの穴の攻略。足場に乗ってスタンバイし、その下にある卵2つの間でヒップドロップ。すると、うまく下まで落ちる。

    ミニバトルやり放題

    マップ選択画面で Select を押しながら L、L、B、A、R と押すと、風船投げ・風船割り・西瓜の種当て・コイン拾いのミニバトルができる。

    高く飛べる

    スイッチを踏むとき A を押しっぱなしにしていると、高く飛べる。

    卵止め

    卵を投げるとき、↑ を押しっぱなしにすると、90度当たりに止められる。

    卵フラワー

    卵フラワーが画面に見える場所でヒップドロップをすると…。

    マップからミニバトルが遊べる

    5-4 のスタート地点左のワンワンのところで、どこでも Pow か、たっぷり「？」雲を使うと…。

    フリーズさせる

    どの画面でもよいので、L + R + Select を同時押し。すると曲は鳴っているのに、画面がフリーズする。カード合わせ、風船投げなどに使える。フリーズしたら、3つのボタンの中の2つまでは離しても大丈夫。すべてのボタンを離すとスリープ状態なので、もう一度 L + R + Select を押して戻す。

    無限 1UP

    1-7 で転がるヤツを咥え、ヘイホーが出る土管まで行く。その横の段差との間にヘイホーが出てくるようにし、そこに転がるやつを入れ放置。しばらく置いておくと999人になる。

    なぜか攻撃してこない

    2-8 でお化けのヘイホーが爆弾を投げてくるが、ヒップドロップをすると、しばらく攻撃しなくなる。

    ごほうびデモ

    6W すべてを1000点満点でクリアする。

    卵を一瞬で補充

    卵を吐き出す花の近くでヒップドロップ。

    赤コインヘイホーの倒し方

    赤コインをもったヘイホーはほとんどが逃げるので、画面に出たらすぐ倒す。蟹は、赤い西瓜だとすぐ倒せる。
