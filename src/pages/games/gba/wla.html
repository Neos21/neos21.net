2001-08-21

ゲーム紹介

相変わらずの壊れっぷりを見せるワリオ。今回もさまざまなアクションを見せてくれる。ゲームの目的は、ボス部屋を開けるための宝石の欠片を集め、「カギのはし」を探し出して連れて帰ること。ミニゲームも入っていて最後まで楽しめる。

ワリオアドバンス ヨーキのお宝 レビュー

ワリオシリーズは攻略に意外と頭を使うことがある。今回もパズル的な要素のあるステージがあり、ひたすら突っ込んでいくだけではダメなのだ。ゲームを買うと付いてくるゴールドシールも、ギリギリなうんちネタで面白いw。

ワリオアドバンス ヨーキのお宝 攻略方法

準備中。

ワリオアドバンス ヨーキのお宝 裏技・小技

敵が…

三日月のゴーストタウンの大きなブロックが2つ並んでいる部屋まで行って、そのブロックとブロックの間に近くにいるピンク色の布を被った敵を入れる。敵が変な動きをする。

持ってないのに持つように見える

赤の通路・-400度の順に進み、キャタピラがあるコースまで行く。浮いているキャタピラに乗り、落ちる瞬間に投げると、手を上げている。

消えたリフト

あやしいこうじょうの最初に出てくる回るリフトを一番右まで動かし、ジャンプせずに左に行くと、段差に当たる。そしたらまた右に行く。リフトが消える。

ワリオホームランで…

9回ホームランを打つとチアガールが出る。激難モードのワリオホームランのピッチャーは、なぜかチアガールらしき人物に変わっている。

簡単なコインの集め方

黄金のピラミッドで最後の通路を選び、すぐゲートに入ると400コイン貯まる。

アイテムでボスに与えられるダメージ

ピンキー

フラワナ

コンダラー

爆弾2、大砲4、白マン4、ラッパ4、黒犬9、大唇8、拳骨12、黒竜7

キャバット

爆弾2、大砲1、白マン3、ラッパ4、黒犬10、大唇7、拳骨9、黒竜8

ヨーキ

爆弾1、大砲1、白マン2、ラッパ3、黒犬8、大唇5、拳骨6、黒竜7

ボスの…

ボスには弱点がある。ミニゲームの10コインで取れるアイテムをうまく使う。青は黒犬、黄は唇、赤はパンチ、緑はドラゴン。これらを使うと、敵のライフを多く減らせる。

カラオケ

全ステージで1万点以上取ると、サウンドルームでカラオケモードが出現する。サウンドルームで Exit にカーソルを合わせ、L + R + ↑ + Start + Select を押すと、カラオケモードが出現する。カラオケが流れている間に L を押すと声なしに、R を押すと声ありになる。

カラオケモードの曲の声・速さが調節できる

カラオケモードで曲が流れているとき、左右で曲の速さ、上下で曲の高さ、低さが調節できる。

エンディング

ボス戦で手に入れた宝箱の数によって、エンディングが微妙に変わる。

オープニングで…

オープニングでダッシュボタンを押すとワリオがダッシュする。

敵の色と強さが変わる

始まりの遺跡の2種類の敵の前でヒップアタックをすると…。

ヒマワリから500コイン

お花畑で、大きなヒマワリに敵を投げると500コイン。

情けなき顔

デブワリオのときに ←・→ を交互に連打すると、ワリオがとてつもなく情けない顔をしている。

激ムズ

難しいモードをクリアすると、新しく激ムズモードが加わる。

微妙に揺れる

アラビアンナイトにある空飛ぶ絨毯を2枚重ねて、その上に上手く乗ると、ワリオが縦に小刻みに揺れる。
