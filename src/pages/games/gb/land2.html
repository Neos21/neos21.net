    ゲーム紹介

    スーパーマリオランドで宇宙人タタンガを退治していたら、今度はマリオ城が「ワリオ」に乗っ取られてしまった。ワリオは、マリオ城に入るために必要な6つの金貨をマリオランドにばら撒いてしまった。6つの金貨を集めて、マリオ城にいるワリオをやっつけよう！

    スーパーマリオランド2 6つの金貨 レビュー

    ワリオが初登場するゲーム。実はワリオのデビューは早かったんだなぁ、なんて思ったりw。ゲームの方も、マリオランドでしか見られない特殊なマリオが見られたりしてとても面白い。ハードが GB なので、グラフィックは他のハードより劣っているが、面白さは他のハードのゲームと比べてもなかなか良いのではないだろうか。

    スーパーマリオランド2 6つの金貨 攻略方法

    準備中

    スーパーマリオランド2 6つの金貨 裏技・小技

    裏面

    1度クリアしたステージに下に行く土管があればできる。その土管に潜っているときに Start を押して Select を押しステージから抜ける。もう一回そのステージに行くといきなり地面に潜る。バグなので止まるかも。

    リセットコース

    ツリーゾーンの蜂のいるところで、スタート地点のところまで蜂を連れてくる。うまく踏めば上のところに行ける。そこで何度かジャンプすると土管に入る音がして上のコースの床の中を右に滑っていく。しばらくすると落ちる (このとき、バニーで右にずっと飛んでいくと幼虫だらけのボーナス面に行け、元のコースに戻れる)。そのまま左に落ちると最初のところのダミーに行ける。ブロックの中はコイン。また下に行くとブロック1個分の大きさの土管や棘、壁の破片が敷き詰められたところに行ける (左右どこへ行こうが無駄)。下に少し行くと OP に戻る。

    土管が

    パンプキンの2つ目のステージの壊せるブロックがたくさんあるところに行く。土管の下にあるブロックを1個壊すと土管に入れないはずなのに、土管の下のところをすり抜ける。しかし結構前に戻るのでメリットはない。

    イージーモード

    最初の土管の場面で Select を押してからファイルを進めると、イージーモードになる。

    ワープ

    マリオ城に行くときに B を押したままにする。画面が止まったあとで B を離す。マリオがスタート地点からマリオ城にワープする。

    残り人数が減らない

    ミスをしたらすぐにポーズをかけ、マップ画面に戻る。

    簡単ボーナスゲーム

    クリアしたステージを始め、中間のベルを鳴らしてからゴールのベルを鳴らす。すぐにポーズをかけてセレクトを押す。するとマップ画面には戻らずボーナスゲームへ行く。再び同じステージを始めると、なぜか中間のベル地点から始まるので、簡単にボーナスゲームができる。

    空気が飛んでいる

    パンプキンの2つ目のステージの中間ベルを鳴らす。そのあと、すぐあとの土管の下に行ってマリオから空気を出す。Start と Select でその面から出る。その面をまたやると空気が飛んでくる。
