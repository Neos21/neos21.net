デザインが同じとの報告

先程、チャットの方に、あるメッセージを頂きました(Thanks:50さん)。

ttp://spr7.com/のTOPページのCSSがここのsiteのポケモンフリークのCSSと同じだと思うのですが…

そのキャプチャが此方です:私のサイト・問題のサイト「ポケ錬道場」

私のサイトがこのデザインにしたのは2005年4月1日です(参考:更新履歴)。そしてこちらのポケ錬道場がこのデザインにしたのは、2005年10月20日、つい最近の様です。

…これだけでは確信が持てないので、CSSファイルを覗いて検証:私のサイトのCSS・ポケ錬道場のCSS

  • 「*」(アスタリスク)の指定から見て行って、大体のスタイル指定が同じ
  • 複数の要素を指定する時、「,」(カンマ)とスペースで区切っている
  • コメントアウトした「=」(イコールの記号)の「=」の数が同じ
  • 所々、書き方の違う指定がある
  • .bg {
     /*
     background-image:url("◆.png");
     background-position:right bottom;
     background-repeat:no-repeat;
     */
     }
    コメントアウトして全く使っていないスタイル部分まで全く同じ

…これで、ポケ錬道場さんが、私のサイトのスタイルを使っている、という事は分かりました。

私はそれをどうするのか

このサイトを開設した当時は、私は更に上のサイトを作りたいと意気込んでおり、様々なサイトを見て良い物を見つけたら、それをそのまま貼って使ってました。上の様に、丸っきり同じという事もありました。

只、一度それで管理人さんに怒られ、「人の物はパクるな」と言われました。確かに、著作権やらの問題がありますから、もしそれを訴えたとしたら、これは著作権法違反となるかも知れません。

しかし、私はあえてこう言います。「パクっても構わないよ。」

人の真似をして何かを得られる事もあります。ウェブデザインに関してもそうだと、私は思います。私も昔はやっていた事ですし、私はパクられても何とも思いません。むしろ、「パクられてしまう程良く出来たデザインを作る事が出来たのか!?」とか思って喜んでしまう程です。

ですので、私はポケ錬道場さんを批判するつもりで日記に書いた訳ではありません。

人の物を真似たり盗用したり、パクるという行為は、大半の人が嫌な気持ちになると言う事を覚えておいてほしいからです。

チャットで報告を下さった50さんも、「このサイトのデザインをパクってるな!?」と思って嫌な気分になったかもしれません。これは又別ですが、「Neo's Worldって、人のサイトのデザインをパクってたの?」と誤解されるのも良くないですよね。

文章がまとめられなくて済みませんが、要するに、私が作った物に関してはパクっても構いませんが、周りの人が嫌な気持ちになるかも知れません。本当は法律違反にも当たりかねない行為ですから、その事を承知した上で行動して下さい。という事です。

私はパクられても何の損もしませんから、パクられても構いません。しかし、「使わせて頂きました」とか何とか、報告が頂けたら良かったかな、とも思いました。

iPodでビデオを見よう! for Windows

先ず、動画を変換するソフトとして、TMPGEncとVideora iPod Converterをインストールします。TMPGEncが必要かどうかは、この後の操作の結果によりますが、ダウンロードするのはフリー版で構いません。Videora iPod Converterは英語です。

インストールしたら、Videora iPod Converterを起動し、左側の「Setup」を選び、「Output Videos to:」で、変換したファイルを置く場所を指定して下さい。そしたら左から「Convert」を選び、「One-Click Transcode」ボタンを押し、変換したいファイルを選びます。後は勝手に変換が始まり、指定した場所に変換されたファイルが保存されています。

ここで、変換した動画をQuick Timeで再生してみましょう。上手く再生出来たら、次の段落は飛ばして読みましょう。上手く再生出来なかった場合は、TMPGEncの登場です(どうやら、Videora iPod Converterはwmv形式等は上手く変換出来ない様です)。

TMPGEncを起動したら、下にある「映像ソース」から、変換したいファイルを選びます。そしたら「ファイル」→「ファイルに出力」→「AVIファイル」と進み、「映像を出力」で「DivXR codec」を選択して変換します。こうして変換されたaviファイルを、Videora iPod Converterで変換させれば、正しく変換されます。

ビデオが変換出来たら、iPodに取り込みましょう。iPodとPCをUSBで繋げ、自動でiTunesが立ち上がるのを待ちます(ここでは2回目以降の接続を前提としているので、初期設定等の説明はしません)。iTunesが立ち上がったら、「ファイル」から「ファイルをライブラリに追加」を選び、変換したmp4ファイルを選択します(Ctrlを押しながら複数選択も出来ます)。取り込みたいファイルをライブラリに追加出来たら、「ファイル」から「iPodを更新」を選び、ビデオをiPodに取り込ませます。これでビデオをiPodに取り込めました!

Windowsの場合、変換が多少面倒ですが、こうして苦労した後にiPodで見るビデオは、PCで見る時とは違って見えるでしょうw。

英語

英語は、ご存知の通り、不規則動詞や「三単元のS」があったりする。

私は統一されていない物が嫌いなので、英語の不規則動詞等は嫌い。

そこで、不規則動詞等を無くして、覚え易い英語を作ってみた。

(中学生なので、文法とかお構いなしに理想を述べています。付き合ってやって下さい。)

三単元・複数形
Sを書くのは面倒なので書かない。複数形は数を表す単語と一緒に使えば良いと思う。「two book」「many book」で通じる気がする。
現在進行形・過去形・be動詞等、単語に何かくっつく奴。
ハイフンで区切ると見易いんじゃね?って事で、どの単語も「○○-ed」「○○-ing」「be-ing」って感じにしちゃおう。
比較級、最上級
「○○er」「○○est」は面倒。全部「more ○○ than」「most ○○ of」(inにはしない)にする。
代名詞
I、my、me、mine。面倒だから全部「I」にしなさい。
aとan
バカ野郎、anなんて要らん。「a」だけにしろ。
液体とかの数え方
無理矢理数えろ。
意味の似ている単語
「some」「any」とか、「mend」「repair」「fix」とか、どれも同じ意味だが使い方が違う単語。こう言うのは要らないから、どれか1つにしろ。あり過ぎて大変だけど。
数詞
one、two、three。first、second、third。面倒臭えな。one、two、three…は数字で絶対に済ませる。first、second、third…は「1th」「2th」「3th」…と、全部thを付ければ楽。
