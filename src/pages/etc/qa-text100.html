    2019年に振り返ってみて

    まずお前はテキストサイトぶってたのかと。…いや、テキストサイトぶってたな。昔の日記を見てもらえば分かるように、「僕の見た秩序。」に多大な影響を受けて (というかモロパクリな文面で) 全然面白くないことを書いてはところどころ文字を大きくしたりして悦に入っていたのだ。「テキストサイトかぶれ」って感じの痛々しさだった。

    当時なんでこんなに痛かったのか、2019年に思い付いた弁解をしておくと、僕は2003年に引越しを経験して、割と本気で自殺を考えるくらい落ち込んでいた。そんな時、落ち込む代わりに打ち込んだのが HTML タグだったのだ。…というのは冗談半分だが。でも、ポケモンのゲームをやって、サイトに攻略情報を書いて、ついでに HTML や CSS も覚え始めて、サイトをキレイに魅せたいなぁと試行錯誤し始めて、Flash アニメとかも作ったりして、という風に、何かやりたいこと・やることを自分で作って忙しくすることで、学校でのイライラを紛らわそうとしていた気がする。

    この質問集に答えたのは2005年で、その当時はそういう負のエネルギーが根源にあったと思う。

    2006年に地元に戻れて、それまでのようなストレスはなくなっていたが、今度は思春期特有の「自己鍛錬に勤しむ気はないけど、何となくモテたいし目立ちたい」みたいな欲求がわずかながら生まれ始めていた。そんな時に「僕秩」を発見し、「俺も面白キャラで行こう」みたいなノリになり、このサイトの日記や mixi なんかでどうでもいいことを大げさに書くようになっていたと思う。

    そうこうしている内に、「何かを書いて発信すること」そのものが趣味みたいになってしまい、今のようなおかしな人間ができあがった次第。

    おかげさまで、大学で周りの人が「レポートが書けない」「字数が稼げない」とか言っている中で、自分は逆に書き過ぎてしまって文字数を削ずに困っていたぐらいだし、会社に入ってもドキュメント作成は (上手とは思わないが) 苦手には感じなくて、「書くクセ」が日頃の役に立っている。それを思うと、あの当時、痛々しいながらもテキストサイトにかぶれておいて良かったのかな？とも思ったりする。
