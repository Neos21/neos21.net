このウェブサイトは何

このウェブサイトは、管理人 Neo の趣味に関するコンテンツを公開しています。主にウェブサイト作成、プログラミング、音楽、映画、ゲームに関するコンテンツを扱っています。

免責事項

このウェブサイトでは、独自ドメイン、サーバレンタル費用を賄うため、Google AdSense、Amazon アソシエイト、楽天アフィリエイトの広告を表示しています。

できるだけ正しいことを書いているつもりですが、中には間違った情報が含まれているかもしれません。このウェブサイトによってどのような不利益が生じても一切の責任を負いかねます。

このウェブサイトの内容に関する指摘やバグなどがありましたら、メールか GitHub Issues にてお知らせいただけると幸いです。
