    Neo's World について

    管理人 Neo の趣味に関するコンテンツを公開するサイト。主に音楽・動画・写真・ゲーム・ウェブサイト作成に関するコンテンツを扱う。

    サイト開設は2002年10月9日。それ以前の2000年前後からウェブサイトは作っており、最近は少ない手打ちの HTML に拘って作成している。

    管理人 Neo について

    ハンドルネーム

    Neo

    (HN を決める時にたまたまテレビで映画「マトリックス」が放送されていた、というだけの理由で付けられた名前。SNS の ID などで「Neos21」という名前を使うのは、「Neo」という名前が既に取得されていることが多く、このサイトの URL「neo.s21.xrea.com」から取って使っているもの。)

    生年月日

    1991年1月11日

    (僕の Amazon ほしいものリストはコチラ)

    趣味
    • ウェブサイト作成 :
      趣味が高じてフロントエンドエンジニアになった。Strict で Valid な HTML・CSS・JS をこよなく愛する。
    • プログラミング :
      趣味のウェブサイト作成を通じて Perl・PHP を眺めていた程度だったが、SE になって Java を学び開眼。VBA・VBScript・Windows Batch・Bash・Ruby・TypeScript などを触る。エンジニアとしてのプロフィール・ポートフォリオは「About Neo」をドウゾ。
    • 音楽鑑賞 :
      ジャズ・フュージョン・AOR 中心。以前は Hip Hop・Acid Jazz・Drum & Bass・2 Step・Post Hardcore などもよく聴いていた。
    • ギター :
      エレキギター。エフェクター含め買い漁る時代があった。ベースも数本所有。最近は落ち着いてスタインバーガーを部屋で爪弾く程度。
    • 映画鑑賞 :
      小さい頃はジャッキーチェンのスタントに憧れ、特撮や VFX のメイキングを見るのも好き。Back To The Future が大好き。
    • 写真撮影 :
      コンデジから始まり、現在は Olympus OM-D E-M10 Mark II と Nikon D5600 を使っている。
    • 画像編集 :
      撮った写真は Photoshop で加工して初めて完成と思っている (つまり腕に自信がない)。
    • 映像撮影 :
      以前はビデオカメラも持っていたが、デジカメ・iPhone で十分な時代になり、ビデオカメラは所有していない。スローモーション大好き。
    • 動画編集 :
      Premiere Pro で動画を編集して YouTube に公開したり。
    • トリッキング :
      Parkour・Free-Running・XMA (MAT)・ブレイクダンスなどをほんのりかじり、トランポリンを飛んだりする。ここ数年は全然運動していなくて全然ダメ。
    • ゲーム :
      ウェブサイトを始めるきっかけになった趣味。任天堂っ子だったため、ポケモン・マリオシリーズが特に好き。コンシューマ機は N64 がほぼ最後で、以降は PC ゲームに移った。最近は時々 PC で GTAV をプレイする程度。
