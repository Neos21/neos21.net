2020-11-01 : サイトをリニューアルしました。合わせて XREA 無料プランから有料プランの XREA Plus に移行し、広告を外しました。さらに、独自ドメイン neos21.net を取得しました。このサイトの元々の URL である neo.s21.xrea.com も残っていますが、コチラは HTTP 通信になります。新ドメイン neos21.net は URL の違いだけで、同じサーバのファイルをレスポンスしていますが、HTTPS 通信に対応しています。

このサイト上でブログも公開したいと思っていますが、現在準備中です。
