毎度忘れるので記事にする。

通常、シェルスクリプトをただ書いただけの時は、

```bash
$ sh ./hoge.sh
```

と書かないと実行できない。コレを

```bash
$ ./hoge.sh
```

だけで実行できるようにするには、`chmod` コマンドで実行権限を付ける。

```bash
$ chmod +x ./hoge.sh
```

__`chmod +x`__。覚えましたし。
