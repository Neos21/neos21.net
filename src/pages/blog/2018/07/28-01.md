Oracle には _`NVL()`_ と __`NVL2()`__ という2つの関数があったので、違いを見てみた。

```sql
SELECT
  NVL(NULL, 'NULL!'),
  NVL2(NULL, 'NOT NULL!', 'NULL!')
FROM DUAL;
```

- _`NVL()`_ は第1引数が NULL の時に第2引数を返すモノ。第1引数が NULL でなければ、第1引数がそのまま返される。
- __`NVL2()`__ は第1引数を評価し、NULL でなければ第2引数、NULL なら第3引数を返す。
