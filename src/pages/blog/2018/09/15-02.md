Bash で直前に実行したコマンドの終了コードを確認するには、__`echo $?`__ という構文を使う。

```bash
# 正しくファイルが削除できた場合
$ rm test
$ echo $?
0

# ファイルが存在せず削除できなかった場合
$ rm test
rm: test: No such file or directory
$ echo $?
1
```

このように。

シェルスクリプトを書いていてコマンドの実行結果に応じた分岐処理が上手く書けなかった時に使った。
