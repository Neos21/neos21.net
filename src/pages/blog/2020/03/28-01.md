Ubuntu に tmux を入れてみる。インストールは以下で OK。

```bash
$ sudo apt install -y tmux
```

罫線が表示崩れを起こしたりしていたのだが、「設定」→「互換性」→__「曖昧幅の文字」を「半角」__にすると崩れなくなった。
