GitHub が色々と Atom フィードを吐けるらしいことを知った。

- [azu/book-review: 本を読んだ感想を書くブログです。](https://github.com/azu/book-review)

azu さんのリポジトリを見ていたら

> RSSで読みたい場合は `https://github.com/azu/book-review/releases.atom` を購読してください

という記載があり、「`github.com` ドメインの Releases ページに `.atom` 拡張子が付いている…！？」と気になったので調べてみた。

- [Githubが結構色々なatomフィードを出せるので、連携で役に立ったりする - Qiita](https://qiita.com/sawanoboly/items/eddc1e230657184d5121)

GitHub リポジトリの Releases、Wiki、Commit などのページは、URL 末尾に `.atom` を付けると Atom フィードが返ってくるようだ。コレを利用すれば、「興味のあるライブラリのリリースを RSS リーダで監視する」といったことも可能なワケだ。ナイス。
