Asus ChromeBook Flip に Crouton で導入した Xubuntu 環境に curl がなかったのでインストールする。

ターミナルを開き、以下のコマンドで OK。

```bash
$ sudo apt-get install curl
```

- 参考：[Ubuntu 10.10にCurlコマンドをインストールする - 大人になったら肺呼吸](http://d.hatena.ne.jp/replication/20110221/1298298256)
