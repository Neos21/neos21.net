面白いライセンスがあった。「Do What The Fuck You Want To Public License」、略して WTFPL。「どうとでも勝手にしやがれクソッタレ・公衆利用許諾 (契約) 書」とでも訳せるライセンス形態。

- 参考：[WTFPL - Wikipedia](https://ja.wikipedia.org/wiki/WTFPL)

面白いので自分が作った [gh-contributions](https://github.com/Neos21/gh-contributions) というシェルスクリプトに使ってみた。

- [GitHub - Neos21/gh-contributions: GitHub の草を生やすシェルスクリプト](https://github.com/Neos21/gh-contributions)
- [gh-contributions/LICENSE at master · Neos21/gh-contributions · GitHub](https://github.com/Neos21/gh-contributions/blob/master/LICENSE)

ちゃんと GitHub でも WTFPL の LICENSE と認識されている。

[![WTFPL](http://www.wtfpl.net/wp-content/uploads/2012/12/wtfpl-badge-1.png "WTFPL")](http://www.wtfpl.net/)
