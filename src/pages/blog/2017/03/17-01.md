PowerShell におけるコメントは、`#` シャープで書ける。

```ps1
# コメントになる。
Write-Host "ほげほげ。"
```

んで、以下のように書くと、__複数行コメント__にできた。

```ps1
<# これは複数行コメント。
ほげほげ。
ふがふが。#>

<#
  インデントしておくと分かりやすいかも。
  ふがふが。
#>

<# 1行でも使えるので、行内にコメントを混じらせることもできる #>
```

Java でいう `/* コメント */` 的な使い方ができる。
