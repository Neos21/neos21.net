**Nim** という言語を見つけたので試してみた。

- [Nim Programming Language](https://nim-lang.org/)

一見すると Python 風のコードだが、静的な型付けができ、コンパイルして実行する形式で C++ 並に速いとか。

Fizz Buzz を出力するサンプルコードを以下のリポジトリに置いた。

- [Neos21/practice-nim: Practice Nim](https://github.com/Neos21/practice-nim?tab=readme-ov-file)

`nimlang/nim` という公式の Docker イメージを使い `nim r` コマンドで直接実行して動作確認した。コレだとホスト環境は汚れないが速度はよく分からない。ｗ

Python っぽい書きぶりがあんまり好きではないが、静的型付けができるので Python 書かされるくらいなら Nim でやりたいな、と思った。
