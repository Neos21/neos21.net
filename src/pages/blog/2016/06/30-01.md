npm でインストールしたライブラリを一覧表示するコマンド `npm list`、エイリアスで書くと `npm ls` だが、そのライブラリが依存しているパッケージが階層表示されるので、見づらい。それを解消する方法。

```bash
$ npm ls --depth=0
```

コレでドウゾ。

`--depth=` で表示する深さを指定できるようだ。グローバルインストールしたものを調べたいときは `-g` オプションを一緒につけて、`npm ls -g --depth=0` で OK。
