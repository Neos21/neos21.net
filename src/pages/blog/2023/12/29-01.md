「*JavaScript ―― You Are Not Java*」というアホな曲を作りました。以下の SoundCloud で聴けます。

- SoundCloud - [025 2023-12-28 JavaScript ―― You Are Not Java](https://soundcloud.com/neos21/025-2023-12-28-javascript-you-are-not-java)

先日買った Cubase Artist 13 で打ち込みをし、Tascam TM-80 (コンデンサマイク) を Yamaha AG06MK2 に繋いで歌を収録。いや、歌と呼べるほどのモノでは…。

> **『JavaScript ―― You Are Not Java』**
> 
> JavaScript　キミは Java とは違うよ  
> Java と JavaScript　その違いを例えてみる
> 
> Java はインド　で　JavaScript はインドネシア  
> Java はメロン　で　JavaScript はメロンパン  
> Java はカレー　で　JavaScript はカレーうどん  
> Java はスズメ　で　JavaScript はスズメバチ
> 
> クラスベース　と　プロトタイプ？　難しいこたぁ分からない  
> でも二人は全然違うよ　その違いを例えてみよう
> 
> Java はハム　　で　JavaScript はハムスター  
> Java はバター　で　JavaScript はバタフライ  
> Java はホース　で　JavaScript はホースラディッシュ  
> Java はカー　　で　JavaScript はカーペット
> 
> キミはブラウザで　私はサーバサイドで生きよう  
> ちょっと待って！　サーバサイド JavaScript もあるんだよ…？
> 
> Java はお爺ちゃん　JavaScript はお兄ちゃん  
> Java は金星　なら　JavaScript は火星かな  
> Java は鉄壁　　　　JavaScript は落とし穴  
> Java は涙　　　　　JavaScript は花　華さ
> 
> JavaScript　キミは Java とは違うよ  
> Java と JavaScript　その違いを例えてみた

作ってみたら、はなわの「佐賀県」みたいな曲調になっていた。「ジャバスクリプト」って文字数が多くて歌詞を作りづらかったのじゃ…。

主に Fedibird でネタを募ったのだが、いただいた元ネタは以下のとおり。皆さんありがとうございました！

- `@AB77ko@fedibird.com`
  - JavaScriptはカレーライスであり、Javaはカレーである
- `@opaupafz2@fedibird.com`
  - JavaScriptは『プロトタイプベース』であり、Javaは『クラスベース』である
- `@mstdn_taki@fedibird.com`
  - javaはおじいちゃんでありjavascriptはお兄ちゃん
- `@nekotako@fedibird.com`
  - 「JavaScript は落とし穴であり」「Java は鉄の処女である」
  - 「JavaScript は花であり」「Java は涙である」
  - 「JavaScript は火星であり」「Java は金星である」
  - 「JavaScript は女子高生であり」「Java は子持ちシングルマザーである」

ネタでも良いから音楽を作っていこう。音を鳴らすのは楽しいぞい。
