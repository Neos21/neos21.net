何度も以下のページを見ている。いい加減覚えろ俺。

- 参考：[JavaScriptで 0, 1, 2, 3, 4 のような連番の配列を生成する方法 - Qiita](https://qiita.com/suin/items/1b39ce57dd660f12f34b)

性能が気になるほど大量の配列を作ることはないので、とりあえず一番短い以下のイディオムを覚えておく。

```javascript
const numbers = [...Array(5).keys()];
// → [0, 1, 2, 3, 4]
```
