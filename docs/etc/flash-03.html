おぶぁうさぎ

当時の文章

著作権フリーといっても放棄はしていない「おぶぁうさぎ」。

この作品で初めて「似ているキャラクター作り」に取りかかった (似てるか？)。このキャラは最初下の図のようだった。 → 「試作品」と alt 属性が振られた 200x200px の GIF 画像が貼られていたが、この画像はサルベージできなかった。

間抜けな顔だが、赤という色はキツい色で、見続けているとムカムカしてきたのでベースの色を使った。さらに丸さをなくし、カクカクにした。作成自体は楽だったが、音声だけは家族に手伝ってもらった。もちろん、あのフレーズだ。

これで完成。ちなみに、完成時のおぶぁうさぎは4号目。

2019年に振り返ってみて

英会話スクール「NOVA」の CM キャラクター「NOVA うさぎ」をパクった Flash。Neo' World 以前のサイトは Outside のパクリみたいな Flash で荒らされて閉鎖に追い込まれたのに、またパクっていた模様。

Windows 標準の「サウンドレコーダー」でボイス録音して、それをスロー再生、倍速再生した音源を利用している。
