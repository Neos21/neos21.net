DreamDaemonRevolution ランキングページ

2003年から2004年にかけて公開していたページを Archive.org からサルベージ。

ココは何？

ココは「TeamFlashSozai」(http://fsozai.hp.infoseek.co.jp/) の (・∀・)yossy◆FlasH.X1/s 様が製作したゲーム Flash、DreamDaemonRevolution (http://fsozai.hp.infoseek.co.jp/DDR/) のスコアランキングを掲載するページ。曲別にスコアの高い順で10位まで掲載する。ランキング登録はページ最下部のフォームから行える。

このランキングページの作成は (・∀・)yossy◆FlasH.X1/s 様から許可を得ているが、(・∀・)yossyFlasH.X1/s 様は一切サポートしてないです。ここで起きた問題等は「DDRランキングスレ」にて報告してくだされ。

ルール：1曲につき1回のスコア登録が可能。それより高いスコアが出た時は、前回のスコアを削除して登録とする。

2019年に振り返ってみて

DreamDaemonRevolution は、2003年当時にハマってやりまくっており、ランキングページでも作ったら面白そうだなーと思って始めた。細々とやっていた割にはランキングの登録があって、1年間くらい置いていた。

ランキングのデザインは2003年の公開当時にテーブルレイアウトで実装していたモノを CSS Grid で再現した。レイアウト用の空 div とかが多くてアレだが、当時みたいに自由にデザインしようと思って作った。一応レスポンシブル対応というか、幅が狭くなったら3カラムを止めて1カラムにするようにしてある。

DDR は大元のサイト「TeamFlashSozai」の変更とともに一時期消滅し、URL を知っている人が直アクセスすれば遊べる、という状況が続いた。その後 Infoseek のレンタルスペースサービス全体が終了し、オリジナルは完全に消滅した。現在でも動画サイトなどでどのようなゲームだったかは閲覧できるが、ホストされているサイトはなさそう。

  • be-interactive.org - DreamDaemonRevolution … HTML ページだけは残っているが、埋め込まれている swf ファイルが 404 になっていて閲覧不可。

2020年には Flash 自体のサポートが終了してしまう。これも一つの思い出として、今回発掘・再公開してみた次第。
