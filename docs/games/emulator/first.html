エミュレータとは

エミュレータとはどんなものなのか、パソコン上で家庭用ゲーム機のゲームを遊ぶ方法など。

エミュレータって何？

「パソコン上で家庭用ゲーム機のゲームが遊べる!!」そんな「エミュレータ」とは、一体何なのだろうか。

エミュレータとは、パソコン上で家庭用ゲーム機の機能を再現 (エミュレート) したモノのことで、ゲームに限らず、パソコンの OS を再現したモノ (Mac 上で Windows を再現する VirtualPC など) もエミュレータと呼ばれる。

現在は、発売されているほぼ全ての機種のエミュレータが開発されており、大体のゲームが、パソコン上で十分に遊べるレベルに達している。

家庭用ゲーム機だけでなく、アーケードゲームもエミュレータが開発されている (MAME など)。

  • 参考 : Wikipedia - ゲームエミュレータ
  • 参考 : Wikipedia - エミュレータ ゲームエミュレータ

エミュレータでゲームをしよう！

エミュレータを使って、パソコン上で家庭用ゲームを遊ぶ方法を紹介する。

必要なものは以下の4つ。

パソコン

まずは当然これが必要。ファミコン程度の古いハードなら大丈夫だが、N64 やプレステなどのハードは、高性能なパソコンでないと処理落ちしてしまうことがあるので、なるべくスペックが高いパソコンを用意しよう。

OS は、エミュレータソフトの対応状況を考えると、Windows がよいだろう。Mac は対応しているエミュレータソフトの数が少ない。

エミュレータソフト

パソコン上で「ゲームソフトの本体」役を務める、エミュレータを用意する。各ゲームのハードごとにエミュレータソフトも異なり、それぞれによってゲームの再現性や機能も変わってくる。

エミュレータソフトはほとんどが無料 (このサイトで紹介しているソフトは全て無料だ)。各エミュレータの詳しい説明は、ハード別エミュレータ解説のページを参照のこと。

ROM ファイル (ROM イメージ)

初めて出てきた言葉だ。これは要するに「ゲームカセット」のことだ。「エミュレータ」が本体、「ROM」がソフト、というワケだ。

この ROM イメージは、自分が持っているソフトから吸い出して用意する必要がある。ファミコンなどのカセット式のゲームソフトは、専用の「吸出し機」を使ってパソコンに吸い出す。

また、ネット上には、著作権フリーの自作 ROM も見付けることができる。

  • 参考 : ねこかぶのホームページ - N64 を吸い出す!! - N64 ROM の吸出し
  • 参考 : Kyu's HomePage (リンク切れ) - GBA ROM の吸出し
  • 参考 : Tittle-Tattle Swop - PD - GBA のフリーゲーム紹介
  • 調べ方 :「ROM 吸出し 方法」「ファミコン ROM 吸出し」など…
BIOS

ハードによっては必要になる。そのゲーム機のエミュレートを行うための核となるデータのこと。エミュレータとセットで用意して「本体」の役割を担う。

これも ROM イメージと同じく、ゲーム機本体から吸出して用意する。

  • 参考 : Irregular Child - BIOS 吸出し - PS・PS2・DC・NDS の BIOS の吸出し方
  • 調べ方 :「BIOS 吸出し」

これらが用意できたら、エミュレータソフトから ROM イメージを開いて、ゲームを起動させてプレイ開始だ。

その他のエミュレータの種類

エミュレータはゲーム機だけではない、という話をしたが、そういったエミュレータを含め、色々なエミュレータの種類を紹介する。

公式なゲームエミュレータ

ゲームの中には、メーカーが公式にエミュレータを使用してゲームを提供していることもある。

代表的なものは、Nintendo Wii などで提供されている「バーチャルコンソール」だ。これは、エミュレータ技術によって N64 以前のゲームをダウンロードして遊べるようになっている。

他にも、PS3 では PS ソフトをエミュレータによって動かしているし、Xbox 360 でも Xbox のゲームをソフトウェアエミュレートで動かしている。

ゲーム機で他のゲーム機をエミュレート

あるゲーム機で、他のゲーム機をエミュレートすることもある。

有名なものとして、DC 上で PS ソフトを動かす「bleem!」や、PSP 上でファミコンソフトを動かす「NesterJ」などがある。

OS 上で他の OS をエミュレート

パソコンの OS をエミュレートするものである。

代表的なのは、Mac 上で Windows をエミュレートする「VirtualPC」や、Windows 上で Mac をエミュレートする「PearPC」などだ。

エミュレートとは違うが、Mac の OS「Leopard」には、Windows をインストールする「Boot Camp」機能が正式に搭載されていたり、逆に Windows パソコンに、ハッキングした Mac OS をインストールする方法があったりする。

エミュレータは違法なのか

よく「エミュレータは違法だ」という話を聞くが、実際はどうなのだろうか。

まずエミュレータソフト自体の違法性。これはない。エミュレートをするソフト自体には何の問題もないのである。

問題なのは、BIOS と ROM イメージの入手方法なのである。

自分が所有しているゲーム機本体とゲームカセットからそれぞれ吸出した場合は、何の問題もない。合法だ。

しかし、最近は UG サイトや P2P などで、誰かが吸出した BIOS や ROM を配布しているようだ。これらをダウンロードして利用するのは、2012年10月から施行された違法ダウンロード法により、処罰の対象になる可能性が非常に高い。

ちなみに、他人の著作物を無断でアップロード・配布するのは完全に違法。つまり、UG サイトの管理者や、P2P でファイルをアップロードしている人は、違法行為を行っているということになる。

吸出し機などを用意するのに、大した労力も費用も掛からないので、違法行為を行うよりは、安全で合法な方法でゲームをするべきだろう。
