ゲーム紹介

愛車 BMW M3 GTR でドライブを楽しんでいた主人公の前に、突如レーザーが現れ、ストリートレースを挑んできた。主人公はレーザーの罠にハマり、愛車を奪われた上に逮捕されてしまう。一方のレーザーは奪った車でストリートレースを勝ち進み、ブラックリストの頂点に君臨する。主人公は愛車を取り戻すため、レーザーを倒すことを決意する…。

映画「ワイルドスピード」シリーズのような世界観で、自車を改造しストリートレースや警察からの逃走を楽しめるゲーム。パソコン以外にも GC、PS2、Xbox 360 などのハードで発売されている。

Need For Speed Most Wanted レビュー

Need For Speed シリーズはこれがほぼ初めてだったが、かなり面白い！シナリオとチャレンジモードクリア後も、警察からどれだけ逃げていられるかで未だに遊びまくっているw。

Need For Speed Most Wanted 攻略方法

NFSMW にはゲームモードとチャレンジモードという2つのモードがあり、両方を完全クリアすることでゲーム達成率 100% になる。ゲーム達成率には関係ないが、クイックレースモード、オンライン対戦モードなどもある。

ゲームモード

シナリオモード。愛車を取り戻すためブラックリスト・ランカーになるのが目標。

さまざまな種類のストリートレースに勝利し、警察追跡中にマイルストンという一定の目標を達成することで、ブラックリスト・ランカーとの対戦が可能になる。

ブラックリスト・ランカーとの対戦に勝利したあとのボーナスマーカーでは、キャッシュや逮捕免除などのマーカーがランダムで1つゲットできるが、その中にライバルカーをゲットできるマーカーがあるので、勝利したあとは必ずこのマーカーを取ろう。その理由は2つある。

  • ライバルカーはショップで施せないペイントが施されており、またそのシナリオの進展度では購入できないパーツが装着されていることもあるため、お金をかけずに速い車をゲットできる。
  • ショップで購入した車は最大で10台までしか所有できないが、ライバルカーはそれとは別にガレージに入れられるため、シナリオクリア後にはブラックリスト・ランカーの15台と足して、25台までガレージに所有しておくことが可能になる。

ライバルカーは一度取り逃すと二度と入手することはできないため、ライバルとの対戦前にオートセーブを OFF にして手動でセーブしライバルと対戦することで、勝利後のボーナスマーカーでライバルカーのマーカーをゲットできなかったときにやり直せるようにしておこう。

ブラックリスト・ランカー #1 のレーザーを倒すと、愛車 BMW M3 GTR に乗って警察追跡からの逃走が始まる。しばらく逃げていると協力者のミアから逃走ルートが指示されるので、橋を大ジャンプして逃げ切ろう。橋を飛び越えるとエンディング、シナリオクリア。BMW M3 GTR はガレージに入っている。

チャレンジモード

全部で 68 + 1 のミッションがある。指定の車で一定の条件をこなすとクリアとなる。

69個目の「バーガーキングチャレンジ」はチートコードによって出現させる。タイトル画面で「burgerking」と入力するとメッセージが出てくる。一度クリアすると再挑戦はできない。

Need For Speed Most Wanted 裏技・小技

チートコード

パソコン版で使えるチートコードは2つ。両方ともタイトル画面でキーボードより入力すると、チートコードを入力したという旨のメッセージが出る。

  • burgerking : チャレンジモードに「バーガーキングチャレンジ」が出現する。
  • castrol : アーケードモードで Castrol Syntec Ford GT が使えるようになる。
