ポケットモンスターファイアレッド・リーフグリーンについて

発売日

2004-01-29

ジャンル

RPG

値段

ソフト単品版 : 3,800円

ワイアレスアダプタ同梱版 : 4,800円

ゲーム紹介

GB の名作「ポケットモンスター赤・緑」に、さまざまな新要素を盛り込んでリメイクされたのが、この「ポケットモンスターファイアレッド・リーフグリーン」。あのときのポケモン達と、新しくなった冒険の世界へ。

ポケットモンスターファイアレッド・リーフグリーン レビュー

基本的なマップやストーリーはほとんど原作と同じだが、B ボタンで走れるなど、ルビー・サファイアから受け継いだ部分も多い。後半で出てくるナナシマなど、原作にはない部分もあって、かなり楽しめる。

ポケットモンスターファイアレッド・リーフグリーン 攻略方法

マサラタウン

  • 主人公とライバルの名前を決める。
  • 家から出て上の草むらに入ると、オーキド博士がやってくる。
  • 研究所でオーキド博士からポケモンをもらう。
  • ライバルとバトルする。ライバルは必ずコチラが不利な相性のポケモンを選ぶ (以降ライバルの最初のポケモンを X とする)。
    • フシギダネ Lv5 を選んだら、ヒトカゲ Lv5 を出してくる。
    • ヒトカゲ Lv5 を選んだら、ゼニガメ Lv5 を出してくる。
    • ゼニガメ Lv5 を選んだら、フシギダネ Lv5 を出してくる。
  • 1番道路を進み、トキワシティへ。

トキワシティ

  • フレンドリィショップ (以降ショップ) に行き、おとどけものを預かる。
  • マサラタウンに戻る。

マサラタウン

  • 研究所に行き、オーキド博士におとどけものを渡す。
  • ポケモン図鑑とモンスターボール5個がもらえる。

トキワシティ

  • 上の方にいるお爺さんに話しかけポケモンの捕まえ方を教えてもらう。このときおしえテレビをもらう。
  • 2番道路 → トキワの森と進み、ニビシティへ。

ニビシティ

  • ジム : トレーナーが1人いるが迂回可能。ジムリーダータケシと戦う。
    • イシツブテ Lv12、イワーク Lv14。
    • フシギダネは Lv10 で覚える「つるのムチ」を使えば楽勝。
    • ヒトカゲは Lv13 まで育てて「メタルクロー」を使うか、トキワの森でキャタピーを捕まえ、「ねんりき」を覚えさせるまで育てて使うといい。
    • ゼニガメは Lv7 で覚える「あわ」か、Lv13 で覚える「みずでっぽう」を使えば楽勝。
  • バトル後に技マシン39 (がんせきふうじ) がもらえる。
  • ニビシティ東の3番道路付近で、助手からランニングシューズがもらえる。B ボタンで走れるようになる。
  • 3番道路 → 4番道路 → おつきみやまと進み、ハナダシティへ。

ハナダシティ

  • 北の24番道路付近でライバルとバトル。
    • ピジョン Lv17、X Lv18、ケーシィ Lv16、コラッタ Lv15。
  • 勝つとボイスチェッカーがもらえる。まずは24番道路へ。

24番道路

  • ゴールデンブリッジを制覇し、25番道路へ。

25番道路

  • 奥にある家でマサキを助け、ふねのチケットをもらう。

ハナダシティ

  • ジム : トレーナーが2人いるが 1人は迂回可能。ジムリーダーカスミとバトル。
    • ヒトデマン Lv18、スターミー Lv21。
    • フシギダネは「つるのムチ」や「はっぱカッター」で楽勝。
    • ヒトカゲはきつい。「えんまく」を覚えさせて命中率を下げて戦うか、トキワの森のピカチュウや24番道路のマダツボミを使うといい。
    • ゼニガメは同タイプなのでダメージは少ないだろうが、同タイプなのでこちらのダメージも少ない。トキワの森のピカチュウや24番道路のマダツボミを使うと楽。
  • バトル後に技マシン03 (みずのはどう) がもらえる。
  • 泥棒に入られた民家から裏口に出て、ロケット団を倒す。
  • 倒すと技マシン28 (あなをほる) をもらえる。
  • 5番道路 → 地下通路 → 6番道路と進み、クチバシティへ。

クチバシティ

  • ふねのチケットを見せてサントアンヌ号に乗る。

サントアンヌ号

  • ライバルとバトル
    • ピジョン Lv19、X Lv20、ラッタ Lv16、ユンゲラー Lv18。
    • ここまでくれば自分の方のポケモンもそこそこ強くなってくるので楽だろう。ユンゲラーは打撃攻撃で倒すといい。
  • 船長の背中をさすり、ひでんマシン01 (いあいぎり) をもらう。

クチバシティ

  • ジム : トレーナーが3人いるが迂回可能。スイッチを押して扉を開けよう (スイッチの法則は分からない)。ジムリーダーマチスとバトル。
    • ビリリダマ Lv21、ピカチュウ Lv21、ライチュウ Lv24
    • 3匹ともあまり相性がよくないので、クチバシティ東の11番道路付近にあるディグダの穴でダグトリオ (出現率低) を捕まえれば楽勝。
  • バトル後、技マシン34がもらえる。ディグダの穴を進み、2番道路のゲートへ。

2番道路

  • ゲート内にいる助手からひでんマシン05 (フラッシュ) をもらう (10種類以上のポケモンを捕まえておくこと)。
  • ディグダの穴 → クチバシティ → 6番道路 → 地下通路 → 5番道路と戻り、9番道路 → 10番道路と進んでイワヤマトンネルへ。

イワヤマトンネル

  • 真っ暗なので、先ほどもらったひでんマシン05 (フラッシュ) を使うとよい。
  • 10番道路へ出たら、シオンタウン → 8番道路 → 地下通路 → 7番道路と進み、タマムシシティへ。

タマムシシティ

  • ゲームコーナーのロケット団を排除後、地下へ。
  • ロケット団リーダーサカキとバトル。撃破後シルフスコープ入手。
  • ジムリーダーエリカ撃破。
  • 北にあるマンション1階のお婆様から「おちゃ」を入手。ヤマブキシティへのゲートを開ける。
  • シオンタウンへ戻る。

シオンタウン

  • ポケモンタワーに潜入。途中ライバル登場。
  • 最上階にいるフジ老人を救出、ポケモンの笛を頂く。
  • タマムシシティまで戻り、左の16番道路へ。

16番道路

  • 上の細い木を切り奥に進み、ひでんマシン02 (そらをとぶ) をもらう。
  • カビゴンを笛で叩き起こし、取り除く。
  • 自転車で17番道路 → 18番道路を進み、セキチクシティへ。

セキチクシティ

  • サファリゾーンの最深部で、金のいればとひでんマシン03 (なみのり) を入手。
  • 金のいればをサファリゾーン園長に渡す。お礼としてひでんマシン04 (かいりき) を頂く。
  • ジムリーダーキョウを撃破。ヤマブキシティへ。

ヤマブキシティ

  • シルフカンパニーへ進入。最上階へ。途中ライバル出現、撃破。
  • ライバルのいた場所周辺に怪しげな人からラプラス入手。
  • 最上階でサカキとバトル。撃破後、社長からマスターボールを頂く。
  • ジムリーダーナツメ撃破。
  • マサラ、またはセキチクから波乗りでグレンタウンへ。

グレンタウン

  • ポケモン屋敷で鍵を入手。
  • 鍵を使い、ジム内へ進入後、ジムリーダーカツラ撃破。
  • ジムリーダー撃破後、マサキが現れ、1の島へ連行。

1の島

  • マサキの友人に会う。
  • そのあと、2の島のゲームコーナーへお届けもの。

2の島

  • お届けモノを渡そうとすると、3の島へ行って人を探せと頼まれる。

3の島

  • 目的の人捜索中、暴走族に絡まれるが撃破。
  • きのみのもりで目標の人物発見。2の島へ連れて帰る。

2の島

  • 連れて帰ったあと、お届けものを渡し、頼まれごと終了。

1の島

  • マサキに報告。以後自由に島と行き来できるようになる。
  • トキワシティへ戻る。

トキワシティ

  • ジムリーダーサカキを撃破。

22番道路

  • ライバルとバトル。

23番道路

  • バッジを見せて通っていく。

チャンピオンロード

  • トレーナー撃破しながら進む。抜けたあとはセキエイ高原。

セキエイ高原

  • 四天王、ライバルを撃破し、殿堂入り。

クリア後の動き方

  1. ポケモン図鑑を60種以上集め、オーキドに図鑑を全国版にしてもらう。
  2. 1の島の「ともしびやま」で「ルビー」入手。
  3. 「ルビー」をニシキに渡し、「レインボーパス」をもらう。
  4. 4の島のいてだきの洞窟で滝登り入手、ロケット団を倒す。
  5. 6の島の点の穴の扉の前でいあいぎりをして中に入り、上、左、右、下と行き、「サファイア」を発見。
  6. 5の島のロケット団倉庫で「サファイア」入手。
  7. 1の島のニシキに「サファイア」を渡せばネットワークマシンが完成。
  8. ルビー・サファイアとの通信が可能になり、ハナダの洞窟でミュウツーに会えるようになる。
  9. シナリオクリア。

主人公の姿

FRLG と RS で通信する。RS から見ると、FRLG のキャラは、RS の主人公の姿になっている。FRLG では、RS の主人公の姿だが、デザインが微妙に違う。

主人公が交通法違反している

サイクリングロードに入り、そこでそらをとぶ。サイクリングロードでは降りてはいけないはずなのに主人公がサイクリングロードから降りている。

サントアンヌ号が出ない

サントアンヌ号の船長にひでんマシン01をもらう。船内のトレーナーに負ける。サントアンヌ号が出ていない状態で物語を進められる。

今回もあった謎のトラック

サントアンヌ号が出ないようにストーリーを進める。波乗りを覚えサントアンヌ号入口の横の海で波乗りをする (右側)。そうするとトラックがある。どういう意味かはわからない。

グラフィカルなポケモンの状態

ポケモンの状態を見ると、通常はポケモンが跳ねるが状態異常だと跳ねず、ダメージがあると少ししか跳ねない。グラフィカルにポケモンの状態が分かる。

説明が出る

道具でタウンマップを開く。すでに行ったダンジョンにカーソルを合わせ A。その入口の画像と共に説明が出る。

音楽

ゲームで流れる音楽の中にポケモンの鳴き声が入っている。トキワの森ではビードルの鳴き声が聞こえた。

サイクリングロードの小技

サイクリングロードで自転車に乗って坂を下っているとき ↓ を押すと少し速くなる。

自己再生技

シャワーズとスターミーでダブルバトルする。シャワーズの HP が危なくなったら、シャワーズにスターミーがハイドロポンプ。シャワーズは回復。スターミーならすばやさも高く、自己再生も覚えるし最適かと。

たまごを育てる技

まずサイクリングロードに行く。あとはタマゴを持ちながら坂を下るだけで約10分前後で生まれる。バトルサーチャーの電池もたまり一石二鳥。

最強はツボツボ? ツボツボ最強講座

ツボツボはとにかく防御、特防が高い。ということで努力値はその2つを中心に上げていく。技はどくどく、まもる、あなをほる、ねむるにしよう。もたせるのはたべのこし。対戦に入ろう。最初にどくどく。次からはひたすら、まもる、あなをほるを繰り返す。すると3ターンに1度しかダメージは受けない。しかもツボツボの防御、特防の高さであまりダメージは受けないし、たべのこしで回復し続ける。危なくなったらねむるで回復。毒が効かない相手はダメかも。

エレベーターの作動時間

今作ではデパートやアジトにあるエレベーターは、移動する階層に比例してエレベーターが作動する時間も変化するようになった。

トキワジム制覇後にタマムシのロケット団アジトに行く。ロケット団が皆消えていてサカキの部屋にネットボールとネストボールが落ちている。

プロフィール作成の小ネタ

プロフィールを作るとき「ふしぎ できごと わくわく ドキドキ」と入力すると「何か聞いたことがあるわ」という。おそらく女の子の従兄弟は、ホウエン地方のトウカシティのポケモンセンターにいる男の子だと思われる。

ワイヤレスアダプタを接続していると、ワイヤレスアダプタを接続してないときに比べ1.9倍電池の消耗が激しい。

やるきについて

特性「やるき」は眠らなくて一見便利な技のようだが、自ら「ねむる」を使えない。

ヘルプ機能の項目

ヘルプ機能の項目は開いたときによって微妙に変化する。特に戦闘中に開くとタイプ別に相性を教えてくれるので忘れてしまった場合に大変便利。

レベルアップしても

トレーナー戦で、1～4回目のころがるで相手ポケモンを撃破して Lv が上がるようにする。Lv があがり新技を覚えようとするが4ついっぱいで覚えられない。というような状態でころがるを忘れさせたとき、次ターン (まだ相手のポケモンが残っているとき) でたたかうを選択すると、忘れさせたはずのころがるがまだ続行する。未確認だが、みらいよちなども似たような現象が起こるかもしれない。

点字の表示時間

点字の表示が L で選ぶと一瞬だけになる。

タマムシのゲームコーナーの右下にある印刷機でトレーナーカードに手持ちのポケモンを50円で印刷できる。
