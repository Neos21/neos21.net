ポケットモンスターエメラルドについて

発売日

2004-09-16

ジャンル

RPG

値段

ソフト単品版 : 3,800円

ワイアレスアダプタ同梱版 : 4,800円

ゲーム紹介

ポケモンエメラルドは、ポケモンルビー・サファイアをベースに、新たなエピソードや遊び方が追加された新バージョンだ。オダマキ博士からもらったポケモンで、ホウエン地方を旅しよう！

ポケットモンスターエメラルド レビュー

エメラルドでは、ルビーのみ、サファイアのみのイベントをうまく融合してあって、グラードンとカイオーガを両方ゲットできたりして、お得感が味わえる。バトルフロンティアも、それぞれのルールに合ったポケモンを育てる必要があるので、長く遊べるだろう。

ポケットモンスターエメラルド 攻略方法

攻略チャート

  1. スタート : 時計を合わせ、隣の家の2階のモンスターボールを調べ、ライバルと会話。
  2. 101番道路 : オダマキ博士が襲われているのでポケモンを選んで助ける。
  3. 先ほど選んだポケモンをパートナーにして、103番道路でライバルとバトル。
  4. 研究所でポケモン図鑑と「モンスターボール」をもらい、外でランニングシューズをもらう。
  5. 101番道路 → コトキタウン → 102番道路 → トウカシティと行き、ジムでセンリと話し、ミツルの世話をする。
  6. トウカの森でアクア団とバトルし、デボン研究員を助ける。
  7. カナズミシティ : いあいぎりおやじの家で「ひでんマシン01」をもらい、カナズミジムでツツジを倒す。
  8. 116番道路を通っていき、カナシダトンネルでアクア団を倒し、荷物とピーコちゃんを取り返す。
  9. カイナに戻って荷物を頼まれる。社長にポケナビをもらう。
  10. 104番道路 : ハギ老人に頼んでムロに行き、ムロジムでトウキを倒す。
  11. いしのどうくつ : 奥でダイゴに手紙を渡し、カイナへ。
  12. カイナシティ : 造船所のツガにクスノキの場所を聞き、海の博物館 2F でアクア団と対戦し、クスノキに荷物を渡す。
  13. キンセツシティ : ジム前のミツルと戦い、キンセツジムでテッセンを倒す。ショップ右の家で「ひでんマシン06」をもらっておく。
  14. シダケタウン : カナシダトンネルの岩を「いわくだき」で壊し「ひでんマシン04」を入手。
  15. 111番道路 → 112番道路 → 炎の抜け道 → 113番道路 → ハジツゲタウン → 114番道路と通り、りゅうせいの滝でマグマ団と遭遇する。
  16. 112番道路のロープウェイからえんとつやまへ行き、マグマ団を倒し、いんせきを取り返す。
  17. デコボコ山道を降りていき、フエンタウンでアスナを倒す。ライバルに「ゴーゴーゴーグル」をもらったらトウカシティへ戻る。
  18. トウカシティ : トウカジムでセンリを倒す。そのあとミツルの父から「ひでんマシン03」をもらえる。
  19. 118番道路 → 119番道路と進み、天気研究所でアクア団を倒す。途中のライバルから「ひでんマシン02」をもらえる。
  20. ヒワマキシティ → 120番道路と進み、ダイゴから「デボンスコープ」をもらう。
  21. ヒワマキシティ : デボンスコープでジム前のカクレオンをどかし、ナギを倒す。
  22. 120番道路 → 121番道路 → 122番水道 → おくりびやまと行き、頂上でアクア団とバトル。「マグマのしるし」をもらう。
  23. デコボコ山道からマグマ団のアジトへ行き、最深部でグラードンを起こし、リーダーと対戦。
  24. カイナで港にいるクスノキと話す。アクア団に潜水艇が奪われるので、ミナモへ行く。
  25. ミナモ右の海からアジトへ行き、最深部で幹部と戦う。
  26. トクサネシティ : ジムでフウとランを倒し、宇宙センターでマグマ団を倒す。ダイゴから「ひでんマシン08」がもらえる。
  27. 128番水道からダイビングでかいていどうくつへ入っていき、アクア団リーダーとバトル。カイオーガが目覚める。
  28. 126番水道からダイビングでルネシティへ行き、ダイゴについていく。
  29. めざめのほこら : 最深部でミクリと話し「そらのはしら」と教える。
  30. 131番道路のそらのはしらへ行き、頂上でレックウザを起こしたらルネに戻る。
  31. 争いがおさまっている。ミクリから「ひでんマシン07」をもらい、ルネジムでアダンを倒す。
  32. サイユウシティをたきのぼりで上っていき、チャンピオンロードを進む。
  33. チャンピオンロード : ミツルとのバトルがある。
  34. ポケモンリーグ : 四天王とチャンピオンを倒すとエンディング。
  35. クリア後 :
    • 殿堂入り後、家の1階に降りると、ラティ出現のニュースが見れる。ここで出現するラティをラティアスかラティオスを選べる。
    • カイナかミナモに行くと船に乗れる。
    • 船は2回目以降はバトルフロンティアへも行けるようになる。

バトルフロンティア攻略

行く条件

殿堂入り後、カイナかミナモから船に乗ってエニシダに会うと、バトルフロンティアに行かれるようになる。バトルフロンティアでは、各施設をクリアすると BP (バトルポイント) というものがもらえ、連続でクリアすると BP が増えていく。ポイントは模様替えグッズや技と交換できる。

シンボル

各施設で勝ち抜いていくと登場するフロンティアブレーンに勝利して銀のシンボルを7つ集めると、エニシダからサンのみがもらえる。2回目のバトルで金のシンボルを集めればスターのみがもらえる。

BP でギャンブル

バトルピラミッド下の建物でギャンブルをしている人がいる。銀のシンボルを3つ集めると、男に指定された施設での対戦を申し込まれ、5・10・15BP のいずれかを賭けて対戦できる。

共通ルール

Lv50 までのポケモンが出場できる「レベル50」と、制限なしの「オープンレベル」の2つがある。伝説のポケモン (カイオーガ、グラードン、レックウザ)、同じポケモン、タマゴは参加できず、参加ポケモンは同じ道具を持てない。

バトルタワー
ルール

7人のトレーナーに勝ち抜くトーナメント式。シングルバトル (1対1)、ダブルバトル (2対2)、マルチバトル (タッグを組んで2対2)、つうしんマルチバトル (通信相手とタッグを組んで2対2) の4つの対戦方法がある。

弟子入り

受付左のトレーナーに話しかけると弟子入り志願される。色々と質問され、答えは弟子に影響される。勝利のセリフを教えると、新しい弟子を持てるようになる。

バトルドーム
ルール

16人がトーナメント形式で対戦する。4回勝てば優勝。シングルバトルとダブルバトルがあり、3匹から2匹を選んでバトル。相手の情報が確認できるので、それを参考にパーティを選ぼう。

ドームスーパースターのヒース

5周目のトーナメントの決勝戦 (20回目) は、フロンティアブレーンの1人、ヒースとバトルになる。勝つと銀のタクティクスシンボルと 13BP がもらえる。相手のポケモンはボーマンダ、ラグラージ、リザードン。

バトルパレス
ルール

シングルかダブルのどちらかを選ぶ。7人と対戦するが、ポケモンには入れ替え以外の命令ができない。行動パターンは性格で決まり、技は攻撃技、自分への補助技、相手への補助技の3つ。性格に合う技をよく使い、合わない技はあまり使わず、成功率も低い。HP が半分以下になると、性格と違う技を出すこともある。

性格と戦い方

バトルパレス左の建物の女の子が、戦い方の傾向を教えてくれる。ただ、女の子の言い方は分かりづらいので、表にまとめてみた。つまり、「普段は戦法1をメインに使うが、HP が半分以下になると戦法2をメインに使う」ということ。タイプ2は、その戦法2がないということだ。

パレスガーディアンのウコン

3週の最後 (21回目) でバトル。勝つとスピリットシンボルと 15BP がもらえる。相手のポケモンはクロバット、ケッキング、ラプラス。

バトルアリーナ
ルール

シングルバトルで7人に勝ち抜く。ポケモンは途中で入れ替えできない。1回の戦いにつき3ターン以内で勝負がつかなければ、3つの判定で勝者を決める。引き分けは両者とも負けとなる。

コツ

判定負けを防ぐため、ねむりやこおり状態にならないようにきのみを持たせたりするとよい。

コゴミ

詳細不明。相手のポケモンはヘラクロス、ブラッキー、ヌケニン。

バトルファクトリー
ルール

シングルとダブルの2種類で、7人に勝ち抜く。ポケモンはランダムに用意された6匹から3匹を選ぶ。相手の情報も確認できる。勝つと相手のポケモンと1匹交換できる。

ファクトリーヘッドのダツラ

詳細不明。

バトルチューブ
ルール

3つの道から1つ選びゴールを目指す。14部屋でクリア。

ラッキー！な部屋

バトルならず (バトルにならない)、スタッフ (全回復)、ジェントルマン (1、2匹回復)。

ラッキー？な部屋

野性ポケモン (オープンレベルはサマヨール Lv95、ハブネーク Lv96、ミロカロス Lv96。逃げられる)、腕の立つトレーナー (シングル。勝利後全回復)。

アンラッキーな部屋

シングル (相手は3匹)、ダブル (各1匹ずつ)、1～3匹がまひ or ねむり or どくに (キルリアが異常状態にする)、1～3匹がこおり or やけどに (ハブネークが異常状態にする)。

ヒント集

迷っているか聞かれたときに「こそこそと音がする」というところはラッキー！な部屋。「モンスターの気配がする」というところはモンスターがエンカウントする。

チューブクイーンのアザミ

28部屋目でバトル。勝つとラックシンボルと 11BP をもらえる。相手のポケモンはハブネーク、ツボツボ、ミロカロス。

バトルピラミッド
ルール

3匹のポケモンを連れて7階まで行けばクリア。道具は持てず、代わりにバトルバッグを持って、ピラミッド内に落ちているアイテムを拾って進める。フロアはランダムに変わり、最大8人のトレーナーがいる。トレーナーは倒すとアイテムの場所や進み方を教えてくれる。野性ポケモンも出て、ポケモンやトレーナーを倒すと少し明るくなる。

ピラミッドキングのジンダイ

3週目の最後 (21段目) に出る。勝利後にブレイブシンボルをもらえる。相手のポケモンはレジロック、レジスチル、レジアイス。

ポケットモンスターエメラルド 裏技・小技

トレーナーヒル攻略

トレーナーヒルは111番道路にあり、トレーナーや仕掛け床をくぐり抜け、どれだけ短いタイムで最上階まで到達できるかを競うタイムアタックができる。友達と速さを比べることもできる。「ポケモンバトルカード e+ エメラルド」でトレーナーやフロアのデザインを変えられる。

ルビー・サファイアと同じネタ
  • タイトル画面で、↑ + Select + B を押すと、データの消去ができる。データは消したら戻らないので注意。
  • ゲーム中に、A + B + Start + Select を押すとリセットになる。セーブしていないデータは消えてしまうので注意。
  • ボックスを整理する際に Select を押すと、手がオレンジになり、メニューが出ずに掴める。A を押したままドラッグするとたくさんのポケモンを掴める。
  • 設定でモードを L=A にすると、L が A の機能になり、左手だけでプレイできる。片手プレイ、左手プレイなどという。
  • ショップのアンケートに「ふしぎ できごと わくわく ドキドキ」と入れると「ふしぎなできごと」が出る。
  • いあいぎりで草を刈れる。
ふしぎなおくりものとジョイスポットサービス

フレンドリィショップのアンケートに「ともだち いっぱい ふしぎ つうしん」と入れると「ふしぎなおくりもの」機能が出るのだが、それを選ぶと「ふしぎなニュース」という機能が出て、ジョイスポットでは最新情報をゲットできる。

また、アンケート用紙に「トップ じょうほう おしえて ください」と入れて、ジョイスポットで「ふしぎなニュース」を受信すると、トレーナーとの対戦結果が上位400位までが分かる。さらに、ジョイスポットでユニオンルームに入ると「ジョイスポットトレーナー・大人のお姉さんのエツコ」が登場し、Lv30 以下のポケモン2匹でバトルができる。

ルビー・サファイアの時計バグを直す

エメラルドとルビー・サファイアを通信ケーブルで繋ぎ (紫がエメラルド)、エメラルドを ON にして、タイトルで B と Select を同時押しする。「木の実問題修理プログラム」と表示されたら、画面の指示に従って修理をしていく。

ボルテッカーピチュー

ピカチュウにでんきだまを持たせて卵を産ませると、ボルテッカーを覚えたピチューが生まれる。

エメラルドに出現しないポケモン

アメタマ、アメモース、アサナン、チャーレム、ザングース、ロゼリア、ルナトーンは、エメラルドではゲットできない。

  • リーグ制覇後はジムリーダーと再び戦える。
  • トクサネのロケットの発射するところの裏が通れる。
  • 通信対戦をしたとき2人ともフロンティアパスをもっていればその対戦を記録に残せる。
