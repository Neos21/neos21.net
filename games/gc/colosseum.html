バリューコネクト
価格 : 3980円

ポケモンコロシアム 概要

発売日

2003-11-21

ジャンル

RPG

値段

5,800円

ゲーム紹介

ポケモンコロシアムは、まったく新しい冒険が登場。パートナーのエーフィとブラッキーを連れて、ダークポケモンを操る悪を倒し、ダークポケモンたちを救い出そう！

ポケモンコロシアム レビュー

相手のトレーナーのダークポケモンを奪い取って、ダークポケモンの心を開いていく (リライブ)。今までとはまったく違ったシステムで、最初は戸惑ったが、次第にリライブが面白くなってくる。GBA との通信もできて、さらに楽しめる。

ポケモンコロシアム 攻略方法

ポケモン名は、そのチャートでゲットできるポケモン。

  1. 町外れのスタンド : 一度中に入り外に出ると、バトルになる。
  2. フェナスシティ : 市長の家に行き、市長と話す。闘技場に行き、外に出るとバトル。
  3. 町外れのスタンド : マスターと話し、モンスターボールをもらう。
  4. フェナスシティ : 市長の家でバトル2連戦。マクノシタ。
  5. フェナスシティの入口でバトル。アリゲイツ or ベイリーフ or マグマラシ (どれか1匹)。
  6. 工事現場 : 人と話してすぐに出る。
  7. パイラタウン : 広場でムウマ、ヨルノズク、ヌオー、モココ、ポポッコ、マグマッグがスナッチできる。
  8. パイラコロシアム入口でバトル。オオタチ。
  9. 発電所へ。
  10. 工事現場 : 右上にある歯車をゲット。
  11. パイラタウン : 発電所へ。
  12. パイラコロシアムに出場。4連戦。
  13. シャドー団基地。
  14. シャドー戦闘員とバトル。ヤンヤンマ。
  15. 屋上の小屋でバトル。テッポウオ、マンタイン。
  16. 洞窟前でバトル。ハリーセン。
  17. 洞窟内。アサナン、ノコッチ、チルット。
  18. ミラーボとバトル。ウソッキー。
  19. プラスル発見。ギンザルの家でプラスルゲット。
  20. 占い師と話す。
  21. アゲトビレッジ : ローガンの家へ。
  22. 聖なるほこらでバトル。カポエラー。
  23. ローガンの家、ビルボーの家へ。
  24. ローガンの家でメール受信。
  25. バトル山 : 10戦。エンテイ。
  26. アゲトビレッジ : ローガンの家へ。
  27. なぞの研究所 : メール受信。
  28. パイラタウン : 警察署でろうやのカギ、牢屋内で寝ているヤツからエレベーターキーをゲット。
  29. ミラーボのいた建物のエレベータに乗る。
  30. アンダー : ヤドヤの2階をとおり隣の建物へ。
  31. きょうかパーツを受け取り左のスレッドの家に行き渡す。
  32. 牢屋前で2連戦。レディアン。
  33. シルバから R-ディスクをもらう。
  34. R-ディスクでヴィーナスのいるスタジオへ。ヴィーナスとバトル。スイクン。
  35. ヴィーナスを追いかける。途中のトレーナーとバトル。グライガー、オドシシ、イノムー、ニューラ。
  36. アンダー駅 : 駅の外で地下鉄のカギをゲット。
  37. 研究所駅 : メインゲートキー、U-ディスクゲット。
  38. アンダー駅 : U-ディスクで外へ。
  39. ダークポケモン研究所 : カードキー、地下通路のカギゲット。
  40. 研究所内でバトル。エイパム、ヤミカラス、アリアドス、グランブル、ビブラーバ。
  41. DNA サンプルを解析して出てきたポケモンを扉に入力。
  42. アンダーへ。
  43. ラルガタワーへ。
  44. ID バッジ (あか、みどり、あお、きいろ) をゲット。デリバード、キマワリ。
  45. ID バッジを3個集めるとメール受信。
  46. アゲトビレッジ : ローガンからマスターボールをもらう。
  47. ラルガタワー : 中央でヘラクロス。
  48. スナッチ団とバトル。エアームド。
  49. 6連戦。ミルタンク、アブソル、ヘルガー、トロピウス、メタグロス、バンギラス。
  50. クリア。
