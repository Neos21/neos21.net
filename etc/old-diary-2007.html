メモ:携帯動画変換君で iPod 用の動画を変換する際の iTunes のバージョンについて

長いタイトルになってしまったが、要するに、携帯動画変換君で iPod 向けの動画を変換する時に、 iTunes のバージョンが 7.5 以上だと変換出来ないという問題があるのである。

iTunes 7.5 以上の時に携帯動画変換君で変換をすると、変換作業が 100% になり、 iTunes に取り込ませる所で止まったままになってしまう。

解決方法は、 iTunes の 7.5 以下のバージョンを使用するというだけ。既に 7.5 以上をインストールしてしまった人は、 iTunes を一旦アンインストールして、旧バージョンを再インストールすれば良い。

管理人が試した所、 Windows Vista にて、 iTunes 7.3.1.3 を使えば、携帯動画変換君は正常に動く事が確認出来た。旧バージョンの iTunes をダウンロード出来るページは以下。

  • iTunes for Windows まとめ - ダウンロード
  • この問題についての参考ページ:教えて!goo - iTunes 携帯動画変換君について

そういえば一昨日かな?、3333333アクセスを自分で取ったよ!

Neo です。いつの間にか333万アクセス越えてたんだなぁ…。

WinIE6 で、このサイトのフッターの背景画像がイカれている件について。

管理人のウチはパソコンが2台(家族用と個人用)あるのですが、どちらも OS は Vista なのです。 Vista は IE7 なので、 IE6 で表示確認が出来ていなかったのです。

Browsershots というサイトで、色々なブラウザで表示確認をして貰えるのですが、 WinIE6 での表示確認のみが、確認してくれるパソコンに日本語の言語パックがインストールされていないらしくて、表示確認してくれません。それ以外のブラウザでは、全てのブラウザでほぼ完璧に表示出来ていたのですが…。

ところで、何故 WinIE6 で表示が崩れるのか。

このサイトは、大まかに別けると、このように div 要素で囲んであります。

  • #wrapper
    • #ad - XREA の広告
    • #site - サイト名
    • #list - パンくずリスト
    • #header - ページタイトル
    • #contents
      • #body - この本文の部分
      • #menu - 横のメニュー部分
    • #footer

#contents 内の #body と #menu は、 float を使って段組にしてあります。そして #footer に background で背景画像を指定してあるのですが、この #footer の背景画像が WinIE6 では表示される位置がおかしくなっています(#body 部分にまで背景画像が来ている)。

float の clear の仕方がポイントなのか、それとも、 WinIE6 ではどのように clear したとしても、 float を clear した要素の下にあるブロック要素の背景画像は表示がバグるものなのか、イマイチ分かりません。

こーゆーのって、悩んでる時はとんでもなく悩んでるんだけど、問題が解決すると、「↑こんなに一杯書いて悩んじゃって、バカじゃねえの?プw」ってなるんだよなぁ…。

手っ取り早く直すなら、 #footer の子要素を、また div 要素か何かで囲んじゃって、それに背景画像を指定すれば良いっぽいんだよなぁ。解決方法が見つからないか、面倒臭くなったら、 HTML を書き換えますw。

電動歯ブラシ買ったよ!

Neo です。

今日は自分の思っている事を書き連ねるだけ。まとまりとか関係ないです。

…

自分より程度の低い人間と一緒に居たくない。自分と同等かそれ以上の人間と居たい。自分より程度の低い人間と居ても良い事は何もない。彼等がやる事なす事全てにイライラするし、下らない問題に巻き込まれたりする。自分以上の人間となら、彼等から教わる事もあるだろうし、自分自身が成長して行ける。彼等からしたら、「 Neo はバカだな」と思うのかも知れないけど、それを考慮して付き合ってくれる様な頭の良い人が良い。と言うのは完全に自己中心的な考えなのだろう。自分さえ良ければ、という魂胆が根本にある発想なのだろう。しかし、自分以下の人間と一緒に居るのは、自分はどうしても我慢出来ない。我慢出来る程大人じゃない。

ここでこの問題を解決する方法は2つ。自分が自分以下の人間と一緒に居ても我慢出来る大人に成長するか、自分以下の人間をバッサリ捨てて自分以上の人間とのみ付き合うようにするか。前者は自分を変えれば良いだけではあるが、その「自分を変える」のは結構難しい。今までずっと、自分以下の人間と居る状況を我慢しようとしてきているが、どうにも我慢ならない。しかし、だからと言って、今彼等をバッサリ切ってしまうと、実際は面倒な生活になる。具体的に言うと、高校と言う場所では、グループを組んで行う授業でグループを組めない、すると授業態度が悪いと見なされ、成績に響く。自分以上の人間をしっかり見つけられていない内に、自分以下の人間を切り捨てると、完全に一匹狼になってしまい、結果として、学校の成績が悪くなってしまう。

だから、今は切れない。自分以上の人間を見つけ出すか、自分が大人になるか、それともどちらも達成せずに、学校の成績を取る方法を見つけるまでは、この状況で我慢して行くしかない。じゃあ、具体的にいつまで我慢したら良いのだろう。来年4月のクラス替えで、運よくクソと別のクラスになれて、運よく良い奴と同じクラスになれて、その良い奴と仲良く出来れば良いだろう。しかし、こういう時に限って、俺はクソとまた同じクラスになって、良い奴と巡り会えない不幸が降りかかって来たりする。「こういう時に限って」がよくある俺だから、今回も期待していない。となると、クソが自分以下の大学に行く or クソが大学に行かないでニート確定している間、自分が目標の大学を見つけ、そこに合格出来れば、それ以降クソの存在に悩まされる事はなくなる。

最低でも、大学に入れば、クソに邪魔される事はなくなる訳だ。そしたら次、自分以上の人間とつるんで行くにはどうしたらいいか、だ。まず高校の間は、別クラスでも、自分以上の奴を見つけ、上手く仲良くなれるよう、自分自身が努力して行動していかなくてはならない。話すきっかけ、仲良くなる方法を考えなくてはならない。大学に入っても、基本的には同じなんだろう。仲良くなっていくように、自分から行動して行かないと、良い奴と出会って、仲良くなって行く事は出来ないのだろう。しかし、自分はこの能力が欠けている。人見知りだし、初対面の人とは何を話したら良いのか分からない。これは場数をこなして行く事で、上手く出来る様になるのだろうか。いずれにしろ、何も行動しないままではダメだ。今より良くなる様に、自分から働きかけて、努力しなくてはならない。

クソと離れる事が最初の目標だが、次の目標は、そのクソを泣かせてから別れる事だ。バカのくせに俺の事をなめてる勘違い野郎をぶっ飛ばしたい。どんな事でもいい。クソ共を打ち負かせて、泣くしかないぐらい追い詰めてやりたい。クソをボコボコにしたい。暴力でも何でも良いけど、もっと違う形で打ち負かした方がカッコイイ。「 Neo には適わない」と思わせたい。「土下座して顔に土塗りたくれ」と言ったら素直に土下座して、顔を泥まみれにするぐらい負かしたい。調子に乗ってるクソ共に、全てにおいてクソ以上である事を知らしめたい。その為に、今出来る事は全てやりたい。何を言われても反論出来ない所まで言い返して打ち負かせたい。それでクソ共が勝手に死んでくれれば最高だと思う。

そもそも今俺が居る場所が、初めから俺にとって全く場違いな場所だって事を、100人いたら100人全員に認めさせたい。「確かにお前は場違いな所にいる。可哀想に、何で Neo はそんなクズみてえな場所にいなきゃいけないんだ?」と、皆に思わせるぐらい。「何の間違いでそんな腐った場所に Neo が居るんだ…。もっと良い場所だから、こっちに来いよ」と、周りの人皆が思わず救いの手を差し伸べてくれるぐらい。俺はクズみたいな場所に来てしまったからって、落ち込んでる場合じゃないんだ。ここが場違いだと思っているなら、俺はここが自分にとって場違いな場所である事を証明しなくてはならない。それも、誰が見ても場違いだと分かるぐらいハッキリと見せ付けてやらなくてはならない。

俺は自分の今居る場所が場違いである事を証明し、クソ共をボコボコにしてぶった切り、自分以上の人間がいる、自分の本当の居場所に行かなくてはならない。その為に、頑張るんだ。勉強も、何もかも。今俺の周りで調子に乗ってるマザーファッカー共をぶっ殺してやるんだ。

この半年以上、ゲームを楽しいと感じなくなってきた。新しいハードにも一切興味が出ないし。

サイト製作も、以前ほどやる気が出ない。デザインも全然思いつかなくなってきた。

ただし、ゲームはもう「好き」ではなくなってきてるが、ウェブサイト作成は「好き」の部類に入る。というかこれを俺から取ったら、俺の長所というか、いいとこが無くなるので必死。

…

このサイトは、自己満足でやってるサイトだ。自分が満足しない内容のままサイトを続けたってしょうがない。

じゃあ、コンテンツを全部消すか。

となると、俺お得意の「もったいない病」が発病する。興味が無くなったコンテンツであっても、消すのは勿体無いのだ。

…

今のコンテンツをどうするかは別として、じゃあ、その後このサイトで何をしていこうか。今の自分の趣味や好きなことってなんだろう。

趣味として今やっているのは、音楽を聴くこと、アクロバットをやること。

興味があってやってみたいと思っていることは、Jackassみたいなバカをやること、スケボー、写真を撮ること、ビデオを撮影すること。

好きな音楽についてレビューしてみるか。…これは面倒になりそうなのでやりたくない。アフィリエイトの広告ばっかりのページにもなりそう…。

アクロバット。一応別サイトで動画は公開してる。これをこのサイトに持ち込むか。それは、今あるコンテンツをどうするかによって変わってくるなぁ。

やってみたいこととして、バカをやること、スケボー。ここは、自分ひとりでは出来なかったり、道具を揃えるのにお金がかかったりして面倒なので放置。

写真やビデオを撮る。これは良さそうだ。ビデオはYouTubeにアップしたり。写真は、いいものが撮れたら「写真素材サイト」なんていう風にもなるんじゃないか?

…

もう一つ考えたのは、ブログに移行するということ。

ブログなら、色々なテーマの記事を書いても、記事毎の「カテゴリ」を指定できるから、閲覧者も見たいものにすぐ辿り着ける。

…

あぁ～悩むなぁ～。俺ってなんて優柔不断なんだろう。

…

現在の「Games」内のコンテンツは、ポケモンとマリオの2シリーズのみ。「Web」内のコンテンツは、他の幾つかのサイトを参考にしてまとめた自分のメモ程度の内容しかない。

つまり、まず「Web」のコンテンツは、あって無いようなものである。このぐらいの内容ならすぐ書けるし、消しても問題ない。

しかし「Games」のコンテンツは、消すのはちょっと気が引ける。というのも、ポケモンもマリオも、Yahooに掲載されるほどの内容だからだ(現在、ポケモンの方はURL変更の手続きをしなかったので消されているがw)。

でもなぁ。これからコンテンツを追加していく気も今の所ないし…。

…

長くなってきたし、考えるのがめんどくなってきたので今日はこの辺で終了。まとめると、

  • 趣味が変わって来たので、サイトのテーマを変えたい。
  • 現在のコンテンツについて、「Web」はとりあえず消しても平気、「Games」は消すのは気が引ける。
  • 新しいコンテンツについて、「写真と動画」辺りが、自分の今の趣味に合ってて良いと思う。
  • ブログにしちゃえば、テーマ分けが簡単。

ってところか。
