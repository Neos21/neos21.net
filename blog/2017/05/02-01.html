git status で日本語のファイル名が数字コードで表示された場合、こう直す

git status で変更が入っているファイルを見たときに、日本語のファイル名が数字コードで見えている場合、以下のように直す。

$ git config --global core.quotepath false
  • 参考：git/日本語ファイル名をgit statusなどで表示する方法 - TOBY SOFT wiki
