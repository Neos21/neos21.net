list-style-type: decimal で ul 要素も ol 要素の見た目になる

小ネタ。

ul {
  list-style-type: decimal;
}

コレで ul 要素でも数字のリストになって ol 要素っぽい見た目になる。

  • デモ : Practice UL Counter
  • コード : frontend-sandboxes/index.html at master · Neos21/frontend-sandboxes

そうでない場合は CSS カウンタを使う方法もあるのだが、ul 要素や ol 要素の場合は list-item という名前のカウンタが用意されているので、counter-reset で変数定義しなくても良いらしい。

  • CSS カウンターの使用 - CSS: カスケーディングスタイルシート | MDN
/* この定義を書かなくても良い */
ul {
  counter-reset: list-item;
}

/* list-item をいきなり使用できる */
ul > li::before {
  content: counter(list-item) " : ";
}

以上小ネタでした。
