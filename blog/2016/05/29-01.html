Mac でスクリーンショットを撮る方法

  • 参考 : Mac でスクリーンショットを撮る方法 - Apple サポート
  • 参考 : Mac OS X のスクリーンショットの撮り方 - maruko2 Note.
用途 キー
画面全体 Command + Shift + 3
ウィンドウ指定 Command + Shift + 4 → Space → ウィンドウをクリック
任意の範囲 Command + Shift + 4 → マウスでドラッグ

デスクトップに png 形式で出力される。
