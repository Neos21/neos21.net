Git のコミットを取り消し、ローカルの変更内容を戻す

直前のコミット内容に戻す場合は以下。

$ git reset --hard HEAD

それ以前の変更内容に戻したい場合は以下。

  1. git log で戻したいコミットの ID を確認してコピーする。
  2. git reset --hard (コミット ID) で元に戻せる。

以上。
