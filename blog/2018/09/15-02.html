直前に実行した Bash コマンドの終了コードを見る

Bash で直前に実行したコマンドの終了コードを確認するには、echo $? という構文を使う。

# 正しくファイルが削除できた場合
$ rm test
$ echo $?
0

# ファイルが存在せず削除できなかった場合
$ rm test
rm: test: No such file or directory
$ echo $?
1

このように。

シェルスクリプトを書いていてコマンドの実行結果に応じた分岐処理が上手く書けなかった時に使った。
