Windows と Mac で CLI から CPU 使用率を確認する

Windoows (XP 以降から最新の 10 まで) と MacOS (最新の Big Sur まで) それぞれで、CLI から CPU 使用率を確認する OS 標準コマンドを調べた。

Windows の場合 : typeperf

PS> typeperf -si 5 "\Processor(_Total)\% Processor Time"
  • -si 5 で5秒間隔に実行する

PowerShell でもコマンドプロンプトでも、上のコマンドで行ける。

  • WindowsとLinuxでCPU, HDD, メモリの使用率を調べる方法 - ミッションたぶんPossible
  • Windows10でコマンドからCPU使用率やメモリ使用率を確認する方法！ | Aprico
  • typeperf.exe を使って CPU使用率をcsvで保存 - Qiita

Mac の場合 : powermetrics コマンドを使う

$ sudo powermetrics --samplers smc | grep -i 'CPU die temperature'
  • -i 3000 などとすると取得間隔をミリ秒で指定できる。未指定時のデフォルトは 5000 (5秒間隔) みたい
  • -n 1 で1回だけ取得できる

計測日時が出力されないのがちょっと分かりにくいところかしら。

  • 参考：powermetrics(1) osx man page
